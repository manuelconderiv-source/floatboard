# Floatboard

A lightweight desktop overlay that keeps notes, todos, and image references visible while you work.

Floatboard sits on top of your workspace as a transparent layer with draggable cards, so you never have to switch windows to check a quick note or tick off a task.

## Features

- **Note cards** — quick text notes with inline formatting
- **Todo cards** — checklists with drag-to-reorder items
- **Image cards** — drop in reference images, screenshots, or mockups
- **Always on top** — cards float above all other windows
- **Drag, resize, snap** — arrange cards freely; they snap together when close
- **Card grouping** — drag cards near each other to form groups with shared headers
- **Minimize & lock** — collapse cards to headers or lock them in place
- **Click-through** — empty space passes clicks to the app below
- **Auto-save** — your board is saved automatically and restored on launch
- **Show/Hide** — toggle visibility with `Ctrl+Shift+F` (or `Cmd+Shift+F` on Mac)
- **System tray** — runs quietly in the tray; right-click to show, hide, or quit

## System Requirements

- **Windows** 10 or later (x64)
- **macOS** 10.13 or later (Intel and Apple Silicon)

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+F` / `Cmd+Shift+F` | Toggle show/hide |
| `Ctrl+Z` | Undo last action |
| Right-click card | Context menu (color, duplicate, delete) |
| Right-click canvas | Export / Import board |
| Double-click card title | Rename card |
